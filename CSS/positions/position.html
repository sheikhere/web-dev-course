Static Position

position : static ;
This is the static or default position of an element, we cannot apply top left right bottom properties in the static position
This is the default position of every element or tag.

Relative Position

position : relative ;
In relative position, top left right bottom properties are applied with in the same flow, means with in the Html default layout or normal flow
It takes it's same space, do not take space out of the flow.
It allows an element to be positioned relative to its normal position within the document flow.

Box1 Box2 Box3 Box4 Box5

Absolute Position

position : absolute ;
In absolute position, top left right bottom properties are applied out of the flow, means not in the Html default layout or normal flow.

Making the position absolute of an element we also have make it's parent element position relative, otherwise it understands whole body is it's body.

It takes the space out of the flow, and other side elements take it's place or space.
Removes an element from the normal document flow and positions it relative to its nearest positioned ancestor.

Fixed Position

Element will remain fixed in the absolute position or position you given, even when the user scrolls the page it will remains fixed.
Fixed is same like absolute but it is fixed on the screen not scrollable.

Sticky Position

Element will remain stick in the relative position means in the default layout and default flow, even when the user scrolls the page it will remains stick.
Stick is same like absolute but it is Stick on the screen not scrollable but in the default flow.
You must have to give the value of top:0px; even 0.
