Units

Units are of two types.

  1. Absolute
  2. Relative

  1. Absolute

Absolute units in CSS are fixed measurements that do not change based on external factors such as the screen size, parent element, or browser settings.

  • px
  • in
  • cm
  • mm

  • px

The px (pixel) unit in CSS is an absolute length unit, meaning it represents a fixed size regardless of the screen size or resolution.
It is used for fixed-size elements like Borders, Shadows, Icons and small images, Fixed layout elements.
border : 10px solid black;

  • in cm mm

The in cm mm unit in CSS are an absolute length unit, primarily intended for output devices with a known physical size, such as print media.
While 1 in inch in CSS is technically defined as 96px pixels
While 1 cm centimeter in CSS is technically defined as 37.7px pixels
While 1 mm milimeter in CSS is technically defined as 3.7px pixels


  1. relative

Relative units in CSS specify a length or size relative to another value, such as the parent element's size, the root element's font size, or the viewport dimensions.

  • %
  • vw
  • vh
  • em
  • rem

  1. %

parent > ---------- > cild The percentage unit (%) in CSS is used to define sizes or positions relative to a parent element or the viewport.
Its primary use cases include: responsive Layouts, flexible Sizing, positioning.
if we give height 40% and width 50% and it's parent height is 30px and width is 15px, it will take 40% height of 30px and 50% width of 15px

  1. vw

The vw unit in CSS, which stands for "viewport width," represents 1% of the viewport's width because whole viewport is consist into 100 parts.
It is primarily used for creating responsive designs where elements scale dynamically based on the width of the user's screen or browser window.

  1. vh

The vh (viewport height) unit in CSS is a relative length unit representing 1% of the viewport's total height because whole viewport is consist into 100 parts.
It is commonly used for creating responsive designs where elements need to scale dynamically with the user's viewport height.

  1. em

The em unit in CSS is a relative length unit that is calculated based on the font-size of the element on which it is applied.
When used for properties other than font-size, it refers to the element's own font-size.
When used for font-size, it refers to the font-size of its parent element.

  1. rem

The em unit in CSS is a relative length unit that is calculated based on the font-size of the element on which it is applied.
When used for properties other than font-size, it refers to the element's own font-size.
When used for font-size, it refers to the font-size of its parent element.
