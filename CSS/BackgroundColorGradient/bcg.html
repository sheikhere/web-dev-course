Color

we can apply color in four ways:
  1. Normal
  2. HEXA code
  3. RGB
  4. HSL

Normal

This is the normal way to apply the color, like we use pink orange black
color : pink;

HEXA Code

This is the hexa code way to apply the color
it has three values # ----- ------- --------
redgreenblue
it's syntax starts from # and it's range is 00-ff where o to 9 and a to f.
color : #396b8e;
It also has opacity value represent by alpha it's range is also 0 to 9 and a to f color : #396b8efe.

RGB

This is the rgb way to apply the color
it has three values red green blue #----- ------- --------
redgreenblue
it's syntax starts from rgb and it's range is 0 - 255
color : rgb(110, 90, 20)
It also has opacity value represent by alpha it's range is 0 to 1 color : rgba(110, 90, 20, 0.998)

HSL

This is the hue saturation lightness way to apply the color
it has three values hue saturation lightness hsl(----- ------- --------)
it's syntax starts from hsl and where range of hue is 0-360, while range of saturation and lightness is 0-100%
color : hsl(110, 90%, 20%)
It also has opacity value represent by alpha it's range is 0 to 1 color : hsla(110, 90%, 20%, 0.904)

Gradient

gradient is the combination of more than one color. It applies with the background property. It is of three types

  1. linear
  2. radial
  3. conic

linear gradient

background :linear-gradient(120deg, yellow , orange, pink);
background-repeat: no-repeat;
background-attachment: scroll;

radial gradient

background :radial-gradient(50% 50%, yellow , orange, pink);
background-repeat: no-repeat;
background-attachment: scroll;

conic gradient

background :conic-gradient(yellow , orange, pink);
background-repeat: no-repeat;
background-attachment: scroll
