Selectors

A CSS selector selects the HTML element(s) you want to style.
Like we used div , span, h1, in the Html document and in the external CSS we can style all the div's by just choosing one div in the stylesheet i.e .css file

Types of Selector

  1. Tag Selector
  2. Class Selector
  3. Id Selector
  4. Pseudp Selector

Tag Selector

Tag selector is a simple selector that we used like div tags, h1 tags and a tags, p tags

in .css file we did h1{ color: blue; text-align: right; background-color: blue; }

This h1 is the Tag selector

Class Selector

This is a collective selector we use to style the html document, at multiple place where we want to do the styling
simply we provide class to the html tag and the then use it in the .css file

div class = "wrapper "> h1> this is the h1 tag /h1> /div>
.wrapper{ color: pink; background-color: rgb(0, 0, 0); text-decoration: solid; }
This container is the class tag same class can be used at multiple points for same styling
And this styling is applied through class selector

Id Selector

this styling is through class

This is a unique selector we use to style the html document, at single place where we want to do the styling
simply we provide id to the html tag and the then use it in the .css file

div id = "unique "> h1> this is the h1 tag /h1> /div>
#unique{ color: pink; background-color: rgb(0, 0, 0); text-decoration: solid; }
This unique is the id tag
one id can be used at single point only because it is unique
And this styling is applied through id selector

Pseudo Selector

this styling is through class

This is a unique selector we use to style the html document, at unique place where we want to do the styling
simply we mention tag and it's child, if many then it's number in the list in the .css file

ol> li>-----/li>
li>-----/li>
li>-----/li>
li>-----/li>
/ol>

ul li:nth-child(3){ color: pink; background-color: rgb(0, 0, 0); text-decoration: solid; }

This styling is applied through ID selector but unorder list list no 3 styling is applied through Pseudo selector
